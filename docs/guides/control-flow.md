# Control Flow

## If, Elif, Else
```the
main {
  statement := 1 < 2

  if statement {
  } elif statement {
  } else {
  }
}
```

## Ternary
```the
main {
  statement := 1 < 2
  a := statement ? 'positive body' : 'negative body'
}
```

## Switch
```the
main {
  switch statement {
    case 'value1': {
      break
    }
    case 'value2':
    case 'valu3':
      break
    default:
      break
  }
}
```
