# Enumerations

## Definition and assignment
```the
enum MovieSeries {
  fast_and_furious,
  harry_potter,
  pirates_of_the_caribbean,
  resident_evil,
  transformers
}

main {
  film1 := MovieSeries.fast_and_furious
  MovieSeries film2 = .harry_potter
}
```
