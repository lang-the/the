# {{ page.title }}
If you want to leave in-code comments for your colleges you can do this in
two ways:

**Inline way** (comment that starts with `//` and ends at the end of the line):
```the
// Inline comment
```

**Block way** (comment that starts with `/*` and ends only with `*/`):
```the
/*
Block comment
*/
```
